    PDF를 Markdown으로 변환하세요

    업로드한 PDF를 baidu/Unlimited-OCR 비전-언어 모델이 분석하여, 표와 그림(figure)까지 포함된 Markdown으로 변환합니다. 변환 과정은 아래에서 토큰 단위로 실시간 스트리밍됩니다.

    • 실시간 스트리밍 — 모델이 생성하는 텍스트를 그대로 지켜보기
    • 그림 자동 추출 — 문서 속 figure를 잘라 이미지로 저장
    • 레이아웃 시각화 — 페이지별 검출 박스 오버레이 확인

    왼쪽에서 PDF를 끌어다 놓거나 선택해 시작하세요.
